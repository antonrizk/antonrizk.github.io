About the card section

For this section of the cards i set them up first for mobile so they take full width and display and they are stacked on top of each other. the next break point would be tablet at MD. i gave the cards a display of the half the grid by giving it a 6 so 2 cards can display beside each other. for the large break point i want the card to have a 4 col display. so they show up 3 next to each other.

inside the card i have an image, a link heading, price, and some lorem for item discription, and a button. so make the button full width of the card i simply gave the div a class to display class of d-grid and the button took the full width of it
